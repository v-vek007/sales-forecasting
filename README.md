---Sales Prediction Project---
Using Linear Regression to predict grocery store sales.

Features:
Data cleaning and preprocessing via prep_sales.py.
Predictive modeling using Scikit-Learn.
Virtual environment setup for reproducibility.

How to Run:
Clone the repo: git clone [YOUR_URL]
Create environment: python -m venv .venv
Install requirements: pip install -r requirements.txt
Run preprocessing: python prep_sales.py
Run model: python linear_regression.py
